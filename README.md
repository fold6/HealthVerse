# HealthVerse_homepage
스포츠 매니저(각종 스포츠 경기등록,기록,채팅,커뮤니티 등)앱 헬스버스를 소개하는 반응형 웹 페이지 (디자인 및 퍼블리싱) <br><br>
※ 존재하지 않는 가상의 앱입니다<br>


📌[바로가기](https://fold6.github.io/HealthVerse/)

<br>
<br>

## 기능 및 특징
- 반응형 웹 페이지
- 앱 장점소개 및 다운링크 표기 랜딩팝업 (24시간 보지않기가능)
- 4가지 주요 서비스를 표기한 슬라이드
- 종목 별 소개 상세이미지
- 6종의 경기매너 표기 서브페이지
